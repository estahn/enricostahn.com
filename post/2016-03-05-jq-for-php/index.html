jq for PHP

 Posted on March 27, 2016  |  1 minutes (89 words)  | 

jq is a great tool for processing JSON on the command line. It’s easy to install (e.g. apt-get intall jq) and easy to use.

When we started writing tests for our new REST API i wanted to have an easy way to query the returned JSON responses. The end result is a small wrapper around jq.

You can find the project on GitHub

Since i have revisited our testing strategy and we moved on to using JSON Schema.

There are alternative projects which might be better suited for you.
